## 项目分析（头脑风暴）
### 1. 数字化支付改变人们的消费习惯
+ 快捷支付使得人们金钱管理意识的缺失
+ 跳过凑整、找零的过程淡化了消费行为
+ 消费软件缺少系统化的账单记录，信息不直观

### 2. 改进方式
+ 呼吁人们使用记账软件强化消费过程
+ 结合多方消费软件给予用户一个更直观系统的消费记录

### 3. 题目来源
+ 近年来消费情况变化

### 4. 需求
1. 增删改查
2. 统计信息（饼图，条形图，折线图），可视化
3. 预算
4. 多账本
5. 固定开支/收入（起始时间）
6. 填充开支/收入金
